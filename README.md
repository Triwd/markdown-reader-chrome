# Markdown Reader - Chrome 插件

一个优雅的 Chrome 插件，用于在浏览器中查看 Markdown 文件，支持大纲导航、主题切换和自动更新。

## 功能特性

✨ **Markdown 渲染** - 使用 marked.js 解析，完美支持 GitHub Flavored Markdown  
📑 **大纲导航** - 自动生成文档大纲，支持点击跳转和滚动高亮  
🎨 **主题切换** - 支持亮色/暗色主题  
💾 **设置持久化** - 大纲显示状态和主题偏好自动保存  
📱 **响应式设计** - 适配不同屏幕尺寸  
🔄 **自动更新** - marked.js 库自动检测和更新  
🛡️ **故障恢复** - 多重备份和降级机制，确保功能可用  

## 安装方法

### 方法一：开发者模式加载

1. 下载本项目代码
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹
6. 插件图标将显示在工具栏上

## 使用方法

### 基本使用

1. **自动启用**：访问任意 `.md` 或 `.markdown` 文件时，插件会自动拦截并启用 Markdown 阅读器

2. **大纲导航**：
   - 左侧显示文档大纲
   - 点击大纲项跳转到对应章节
   - 滚动时自动高亮当前章节
   - 点击大纲标题栏的「×」按钮或工具栏的「☰」按钮可隐藏/显示大纲

3. **主题切换**：
   - 点击工具栏的「☀️」按钮切换亮色/暗色主题
   - 主题偏好会自动保存

### 高级功能

1. **自动更新**（v1.1.0 新增）：
   - 点击插件图标打开设置面板
   - 勾选「自动更新」选项（默认开启）
   - 系统会定期检查 marked.js 的最新版本并自动更新

2. **手动检查更新**：
   - 点击插件图标
   - 展开「高级设置」
   - 点击「检查更新」按钮
   - 如有新版本，可选择「立即更新」或「忽略」

3. **故障恢复**：
   - 如果渲染引擎出现故障，插件会自动尝试恢复
   - 如果恢复失败，会显示原始 Markdown 内容
   - 可在设置面板中「重置配置」解决大部分问题

## 技术架构

### 版本管理 (version-manager.js)
- 自动检测插件版本变化
- 配置迁移和向后兼容
- 版本回退处理

### 自动更新 (auto-updater.js)
- 定期检查 marked.js 更新
- 自动下载和验证新版本
- 失败自动回滚到备份版本
- 紧急降级方案

### 错误处理 (error-handler.js)
- 全局错误捕获
- 自动恢复机制
- 降级渲染（显示原始内容）
- 用户友好的错误提示

## 技术栈

- **Manifest V3** - Chrome 插件最新标准
- **marked.js** - Markdown 解析器（本地打包）
- **原生 JavaScript** - 无框架依赖
- **CSS Variables** - 主题切换
- **Chrome Storage API** - 数据持久化

## 浏览器支持

- Chrome 88+
- Edge 88+
- 其他基于 Chromium 的浏览器

## 数据存储

插件使用 Chrome Storage API 存储以下数据：

- `mdReaderConfig` - 用户配置（本地存储）
- `mdReaderVersion` - 插件版本（本地存储）
- `markedJsContent` - 当前使用的 marked.js（本地存储）
- `markedJsBackup` - 备份版本（本地存储）
- `markedJsFallback` - 紧急降级版本（本地存储）
- `mdReaderErrorLog` - 错误日志（本地存储）
- `outlineVisible` - 大纲显示状态（同步存储）
- `theme` - 主题设置（同步存储）

## 故障排除

### 问题：Markdown 文件无法渲染

**解决方案：**
1. 检查插件是否已启用
2. 刷新页面重试
3. 点击插件图标，选择「重置配置」

### 问题：marked.js 更新失败

**解决方案：**
1. 检查网络连接
2. 稍后再试（可能是 CDN 暂时不可用）
3. 手动在设置中关闭「自动更新」

### 问题：大纲不显示或不同步

**解决方案：**
1. 检查设置中的「大纲滚动同步」选项
2. 尝试切换「显示大纲」开关
3. 重置配置

## 更新 marked.js 的方法

### 方法一：自动更新（推荐）
开启自动更新选项，系统会自动检查并更新

### 方法二：手动更新
1. 点击插件图标
2. 展开「高级设置」
3. 点击「检查更新」
4. 如有新版本，点击「立即更新」

### 方法三：手动替换文件
```bash
npm install marked --save
cp node_modules/marked/lib/marked.umd.js js/marked.min.js
```
